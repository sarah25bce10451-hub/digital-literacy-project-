## 📌 Project Overview

This project was completed as part of the Digital Literacy course, where I took on the role of a Student Digital Ambassador. The objective was to build a digital literacy portfolio that helps students understand and navigate the digital world effectively. The project covers multiple aspects such as digital awareness, online safety, professional presence, communication etiquette, and cybercrime awareness. All tasks are organized in this repository along with supporting files, screenshots, and reflections.
